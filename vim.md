## Vim cheatsheet ##

`f`     find character  
`,`     reverse last find  
`;`     repeat last find  

`&`     search word  
`#`     reverse search word  

`A`     append to end of line (a$)  
`I`     insert in front of line (^i)  
`C`     replace to end of line (c$)  
`s`     replace letter (cl)  
`S`     replace line (^C)  
`cc`    replace line  
`O`     insert newline above (ko)  
`.`     repeats last action
