## Vim cheatsheet ##

`cc`    replace line  
`f`     find character  
`;`     repeat last search  
`A`     append to end of line (a$)  
`I`     insert in front of line  
`C`     replace to end of line (c$)  
`s`     replace letter (cl)  
`S`     replace line (^C)  
`O`     insert newline above (ko)  
`.`     repeats last action
